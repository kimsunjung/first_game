# 현재 상태 + 결정된 방향

## 문서 목적
- 다른 PC 또는 새 세션에서 작업을 이어갈 때, 현재 프로젝트 상태와 결정된 방향을 한 장으로 파악하게 한다.
- 아트/에셋 의사결정 맥락, 작업 우선순위, 다음 실행 포인트를 요약한다.

작성일: 2026-04-21

---

## 1. 기술 스택
- Engine: Godot 4.6 (C# / .NET 8)
- 장르: 탑다운 액션 RPG 던전 크롤러
- 플랫폼: 모바일 (Google Play / App Store) 출시 목표
- 목적: 개인 성취 (수익 무관)

---

## 2. 씬 구조
```
town (허브)
  └─ field_1 (오크류 필드)
       ├─ dungeon_1 (오크킹 보스)
       └─ field_2 (스켈레톤 필드)
            └─ dungeon_2 (스켈레톤 킹 보스)
```

### 맵 상태
| 맵 | 상태 | 비고 |
|----|------|------|
| `town.tscn` | `ColorRect` 플레이스홀더 | 기능 오브젝트만 배치 |
| `field_1.tscn` | **타일 기반 (MapGenerator)** | 최근 런타임 타일 생성 추가 |
| `field_2.tscn` | `ColorRect` | 스켈레톤 3종 + 던전2 포탈 |
| `dungeon_1.tscn` | 기본 구조 | 오크킹 보스 |
| `dungeon_2.tscn` | 기본 구조 | 스켈레톤 킹 보스 |

## 4. 아트/에셋 방향 (결정 완료)

### 메인 에셋
- **Franuka 계열 (Complete Fantasy RPG Bundle)** 로 단계적 풀 교체 확정
- 구매 시점: **PC 변경 후 (2026-05월경 맥북 Air M5 전환 시)**

### 선택 이유
- 팩 종류가 다양하고 전체 스타일을 맞추기 좋음
- 단일 작가 → 외곽선/채도/볼륨감 자동 일관성
- 상업적 사용 허용, 모바일 스토어 출시 가능
- 16x16 기반 + 3가지 크기(100%/200%/300%) 제공 → Godot 4.6 + zoom=3x와 호환

### 라이선스 확인 완료
- ✅ 상업적 사용 가능
- ✅ 구글/애플 스토어 출시 가능
- ❌ 에셋 자체 재판매/재배포 금지
- 권장: itch 페이지 크레딧

### 이전 방향에서 변경
- 초기: `Pixel Crawler` 중심 유지 (안전 경로) → archive 이동
- 현재: `Franuka` 풀 교체로 변경 (일관성 우선)

## 5. 작업 우선순위

Franuka 번들 구매 후 실행 순서:
1. **UI** — HUD/인벤토리/상점/스킬/캐릭터창
2. **Player** — `player.tscn` 1차 교체 (성공 여부 = 전체 성공 지표)
3. **Enemy** — `enemy.tscn`, `Resources/Enemies/*.tres`
4. **NPC** — shop/skill_shop/blacksmith
5. **Town** — `town.tscn` 타일 기반 허브
6. **Field_1** — 타일 바꾸고 구역/동선 재정리
7. **Dungeon** — 던전1, 2
