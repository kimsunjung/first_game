## Screen Fit
- [ ] HUD HP/MP/EXP가 화면 밖으로 나가지 않는다.
- [ ] 골드/레벨/경험치 텍스트가 잘리지 않는다.
- [ ] 공격/상호작용/인벤토리/캐릭터/스킬 버튼이 서로 겹치지 않는다.
- [ ] 인벤토리 창이 화면 안에 들어온다.
- [ ] 스킬창이 화면 안에 들어온다.
- [ ] 캐릭터 창이 화면 안에 들어온다.
- [ ] 상점/스킬상점/강화 UI가 화면 안에 들어온다.
- [ ] 긴 아이템명/스킬명이 버튼이나 패널 밖으로 튀어나오지 않는다.

## Touch Controls
- [ ] 가상 조이스틱 이동이 자연스럽다.
- [ ] 공격 버튼을 오른손 엄지로 누르기 쉽다.
- [ ] 스킬 버튼 Q/W/E/R 4개가 오입력 없이 눌린다.
- [ ] 상호작용 버튼 위치가 엄지 동선에 맞는다.
- [ ] 상단 메뉴 버튼들이 너무 작지 않다.
- [ ] 버튼을 연타해도 UI 토글/공격/스킬 입력이 꼬이지 않는다.

## Core Loop
- [ ] 마을에서 필드로 이동 가능하다.
- [ ] 필드에서 적을 처치할 수 있다.
- [ ] 드랍 아이템을 획득할 수 있다.
- [ ] 인벤토리에서 아이템 아이콘이 정상 표시된다.
- [ ] 장비 장착/해제가 정상 동작한다.
- [ ] 소모품 사용이 정상 동작한다.
- [ ] 강화 UI에서 장비 강화가 가능하다.
- [ ] 던전 진입이 가능하다.
- [ ] 보스 전투 진입/처치가 가능하다.
- [ ] 마을 귀환 또는 씬 전환이 정상 동작한다.

## Save / Load
- [ ] 저장 파일 생성이 정상 동작한다.
- [ ] 앱 종료 후 재시작해도 레벨/경험치가 유지된다.
- [ ] 장비 상태가 유지된다.
- [ ] 강화 수치가 유지된다.
- [ ] 인벤토리 아이템과 수량이 유지된다.
- [ ] 습득 스킬이 유지된다.
- [ ] 현재 씬/스폰 위치 복원이 의도대로 동작한다.
- [ ] 이전 세이브 버전 마이그레이션이 실패하지 않는다.

## Performance
- [ ] 일반 필드 이동 중 프레임 드랍이 체감되지 않는다.
- [ ] 다수 적과 전투 중 프레임 드랍이 체감되지 않는다.
- [ ] 다수 드랍 아이템이 있어도 버벅이지 않는다.
- [ ] 씬 전환 시간이 과도하지 않다.
- [ ] UI 창을 열고 닫을 때 끊김이 크지 않다.

## Audio / Feedback
- [ ] 공격/피격/획득/강화 피드백이 구분된다.
- [ ] 모바일 스피커 기준 사운드가 너무 크거나 작지 않다.
- [ ] 진동 또는 추가 피드백이 필요한 지점이 보인다.

- [ ] 계약 포기 후 저장/로드 시 상태 일치.
- [ ] 저장 후 재시작 시 active contract 진행도/완료대기 상태 복원.

- [ ] Rare/Epic/Legendary 드랍 글로우가 희귀도별로 구분되고 펄스가 과하지 않다.
- [ ] 계약 보드가 기존 NPC/포탈 상호작용과 겹치지 않고 동선을 막지 않는다.
- [ ] 보드 UI 텍스트가 행/버튼 밖으로 삐져나가지 않는다(모바일 해상도).

## 대규모 시스템 통합 수동검증 (2026-05-19 — 실제 플레이 순서)

헤드리스 게이트(validate/balance/build/test)는 데이터·로직만 본다. 아래는
Godot 에디터에서 **순서대로** 한 번 밟아야 하는 항목. 한 줄 = 한 확인.

### A. 신규 시작 → 거점(town) 기본
- [ ] 새 게임(클래스 선택) → town 진입, 좌하단 맵 이름 "마을" 표시.
- [ ] 미니맵이 거점에서 **숨김**(town은 단일화면). 사냥터 진입 시 표시.
- [ ] HUD 상단 이펙트 바: 버프/상태 없을 때 비어있고 레이아웃 안 깨짐.

### B. 계약 보드 (town ContractBoardNPC)
- [ ] 보드 열림 → 진행 중 0 / 수락 가능 목록 표시, 텍스트 행 밖 안 삐짐.
- [ ] Kill/Gather/BossKill/Mining 라벨이 신규 문구로 표시
      (Gather = "납품(완료 시 소모)" + "보유 X/필요 Y · 완료 시 소모").
- [ ] 3개까지 수락, 4번째 "가득참" 비활성. 포기 동작.
- [ ] Gather 계약 수락 후 재료 보유 시 진행 자동 반영. 재료를 창고로
      빼면 보드 재오픈 시 "완료"가 사라지고 "포기"로 내려감(stale 없음).
- [ ] Gather 완료 → 재료 정확히 Goal개 소모 + 골드/EXP 지급.
- [ ] 가방 가득 채운 뒤 아이템 보상 계약 완료 → "보류됨, 가방 비우면
      자동 회수" 토스트, 가방 비우면 실제 회수(중복/손실 없음).
- [ ] 계약 완료/포기 후 즉시 세이브(재진입 시 상태 유지).

### C. 창고 / 제작 / 재련 (4거점 전부: town·field_outpost·harbor·mountain)
- [ ] 각 거점에 Storage/Crafting/Reforge NPC 상호작용 가능(겹침/사각 없음).
- [ ] 창고 보관/꺼내기 — 골드·아이템 이동이 세이브에 반영.
- [ ] 제작 레시피 — 재료 소모 + 결과 지급, 가방 가득 시 보류 처리.
- [ ] 재련 — 미장착 장신구만 대상, affix 재생성, enhance_stone 소모.
- [ ] 기능 NPC(storage/crafting/reforge)는 챕터 대사 없이 바로 UI(정상).

### D. 신규 스킬 (스킬샵에서 학습 후)
- [ ] ChainLightning: 첫 타깃이 화면 안에서 잡히고 인접 연쇄(최대 5),
      먼 점프가 끊겨도 화면 밖 적을 시작점으로 안 잡음(240px).
- [ ] LifeDrain: 흡혈량이 방어/저항 높은 적·체력 1 적에서 과다하지 않음
      (실제 적용 피해/3).
- [ ] FlameWave/FrostNova/RainOfArrows/ArcaneMissile/PiercingShot 화면
      밖 적을 자동 타격하지 않음.
- [ ] 스킬샵 클래스 탭 필터가 스킬북을 올바른 직업 탭에 노출(R15 정합).

### E. HUD 이펙트 바 / 미니맵
- [ ] 상태이상/버프 발생 시 상단 아이콘 표시, 클릭 시 목록+남은시간 팝업,
      잔여<3s 깜빡임, 바깥 탭으로 닫힘(미니맵/조작과 터치 간섭 없음).
- [ ] 씬 전환 직후(플레이어 freed 타이밍) 이펙트 바 접근 크래시 없음.
- [ ] 미니맵: 적 점이 폰에서 보일 정도(2.2px), 플레이어/포탈 식별 가능.

### F. 사냥터 드랍 / 파밍 루프
- [ ] town_region 적 처치 시 orc_leather가 체감상 가끔 떨어짐(≈0.06),
      후반 권역일수록 테마 재료 더 희소(0.04/0.03/0.02).
- [ ] 권역 테마 재료 드랍이 기존 핵심 드랍 빈도를 줄이지 않음(독립 굴림).
- [ ] mine_3 광맥 채광 → crystal_ore, Mining 계약 진행/완료 가능.

### G. 신규/이동 NPC 좌표
- [ ] 4거점에서 NPC/포탈/SavePoint가 서로 가려 상호작용 불가한 곳 없음.
- [ ] (알려진 cosmetic) town ShopSign 노드 중복(동일 좌표 2개) — 별도
      정리 작업으로 분리됨. 기능 영향 없음 확인.
