# Mobile QA Checklist

## 목적
- 모바일 출시 목표 기준으로 실제 기기/모바일 해상도에서 깨지는 문제를 빠르게 찾는다.
- 각 Phase가 끝날 때 같은 항목으로 회귀 검증한다.
- PC 에디터에서 보이는 문제와 실제 모바일 플레이 문제를 분리한다.

## Screen Fit
- [ ] HUD HP/MP/EXP가 화면 밖으로 나가지 않는다.
- [ ] 골드/레벨/경험치 텍스트가 잘리지 않는다.
- [ ] 공격/상호작용/인벤토리/캐릭터/스킬 버튼이 서로 겹치지 않는다.
- [ ] 인벤토리 창이 화면 안에 들어온다.
- [ ] 스킬창이 화면 안에 들어온다.
- [ ] 캐릭터 창이 화면 안에 들어온다.
- [ ] 상점/스킬상점/강화 UI가 화면 안에 들어온다.
- [ ] 긴 아이템명/스킬명이 버튼이나 패널 밖으로 튀어나오지 않는다.

## Touch Controls
- [ ] 가상 조이스틱 이동이 자연스럽다.
- [ ] 공격 버튼을 오른손 엄지로 누르기 쉽다.
- [ ] 스킬 버튼 Q/W/E/R 4개가 오입력 없이 눌린다.
- [ ] 상호작용 버튼 위치가 엄지 동선에 맞는다.
- [ ] 상단 메뉴 버튼들이 너무 작지 않다.
- [ ] 버튼을 연타해도 UI 토글/공격/스킬 입력이 꼬이지 않는다.

## Core Loop
- [ ] 마을에서 필드로 이동 가능하다.
- [ ] 필드에서 적을 처치할 수 있다.
- [ ] 드랍 아이템을 획득할 수 있다.
- [ ] 인벤토리에서 아이템 아이콘이 정상 표시된다.
- [ ] 장비 장착/해제가 정상 동작한다.
- [ ] 소모품 사용이 정상 동작한다.
- [ ] 강화 UI에서 장비 강화가 가능하다.
- [ ] 던전 진입이 가능하다.
- [ ] 보스 전투 진입/처치가 가능하다.
- [ ] 마을 귀환 또는 씬 전환이 정상 동작한다.

## Save / Load
- [ ] 저장 파일 생성이 정상 동작한다.
- [ ] 앱 종료 후 재시작해도 레벨/경험치가 유지된다.
- [ ] 장비 상태가 유지된다.
- [ ] 강화 수치가 유지된다.
- [ ] 인벤토리 아이템과 수량이 유지된다.
- [ ] 습득 스킬이 유지된다.
- [ ] 현재 씬/스폰 위치 복원이 의도대로 동작한다.
- [ ] 이전 세이브 버전 마이그레이션이 실패하지 않는다.

## Performance
- [ ] 일반 필드 이동 중 프레임 드랍이 체감되지 않는다.
- [ ] 다수 적과 전투 중 프레임 드랍이 체감되지 않는다.
- [ ] 다수 드랍 아이템이 있어도 버벅이지 않는다.
- [ ] 씬 전환 시간이 과도하지 않다.
- [ ] UI 창을 열고 닫을 때 끊김이 크지 않다.

## Audio / Feedback
- [ ] 공격/피격/획득/강화 피드백이 구분된다.
- [ ] 모바일 스피커 기준 사운드가 너무 크거나 작지 않다.
- [ ] 진동 또는 추가 피드백이 필요한 지점이 보인다.
