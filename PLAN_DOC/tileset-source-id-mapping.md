## 맵 생성에 주로 사용할 Source ID 요약

### 바닥 (GroundLayer)
- **0**: Floors_Tiles.png (풀, 흙, 돌바닥 등 메인 바닥)
- **1**: Dungeon_Tiles.png (던전 구역)
- **38**: Buildings/Floors.png (건물 내부 바닥)

### 장식 (DecorationLayer)
- **21**: Vegetation.png (풀, 꽃)
- **25**: Shadows.png (나무/건물 그림자)

### 장애물 (ObstacleLayer) - 충돌 있음
- **4**: Water_tiles.png (물)
- **5~19**: 나무 (Model 01/02/03, 다양한 크기)
- **22**: Rocks.png (바위)

### 구조물/건물
- **38~42**: 건물 (바닥, 벽, 지붕, 소품, 그림자)
- **43~92**: 각종 스테이션 (연금술, 모루, 모닥불, 요리, 용광로, 제재소, 작업대)
